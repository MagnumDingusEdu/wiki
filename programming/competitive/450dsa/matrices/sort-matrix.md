# Sorted matrix
You are given an NxN matrix. Sort all elements of the matrix.

## Examples
```cpp
Input:
N=4
Mat=[[10,20,30,40],
    [15,25,35,45] 
    [27,29,37,48] 
    [32,33,39,50]]
Output:
    10 15 20 25 
    27 29 30 32
    33 35 37 39
    40 45 48 50
```

## Constraints
Expected Time Complexity: `O(N^2LogN)`
Expected Auxillary Space: `O(N^2)`

```cpp
1<=N<=1000
1<=Mat[i][j]<=105
```
